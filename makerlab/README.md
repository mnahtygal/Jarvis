# Jarvis Maker Lab

Maker Lab contains the physical hardware, printable components, fixtures,
calibration tools, electronics, and documentation developed for Jarvis.

## Design Principles

- Parametric and reusable designs
- Version-controlled hardware
- Modular camera and sensor mounting
- Documented print settings and assembly
- Replaceable and reproducible components

## Current Projects

- Vision Tower
- ASA tolerance and calibration tools
- Camera mount system
- Scan Mat fixtures
- Lighting and electronics integration

## Directory Overview

- `calibration/` — printer and material calibration models
- `vision_tower/` — Vision Tower CAD, exports, BOM, tests, and documentation
- `camera_mounts/` — interchangeable camera mounting systems
- `scan_mat/` — fixtures and alignment tools
- `electronics/` — lighting, power, USB, and wiring designs
- `tools/` — shared OpenSCAD modules, scripts, and templates
- `archive/` — superseded or experimental designs

## Gold Repository

The Thor copy of the Jarvis repository is the authoritative source for
Maker Lab hardware development.
