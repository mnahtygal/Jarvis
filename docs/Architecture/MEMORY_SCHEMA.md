# Memory Schema

## Purpose
Define how Jarvis stores durable engineering knowledge.

## Memory Types
- User preferences
- Project memory
- Conversation summaries
- Scan history
- Device inventory
- Engineering notes

## Storage
PostgreSQL + pgvector

## Principles
- Explicit writes
- Inspectable records
- Versionable schema
- No duplicate vector stores

## Future
Per-project namespaces and memory lifecycle management.
