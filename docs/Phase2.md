# Phase 2: Maker & Workshop Assistant

Phase 2 turns Jarvis into a practical engineering partner.

## Main Goal

Jarvis should help with real workshop tasks:

- scanning parts
- measuring objects
- generating starter CAD
- tracking projects
- assisting with 3D printing
- preparing laser engraving workflows
- helping with electronics and Raspberry Pi projects
- supporting SDR experiments
- assisting software and data engineering tasks

## Current Priority

The current priority is Scan Mat Mode.

The immediate goal is to reliably capture, process, display, and store scan outputs.

## Phase 2 Sequence

1. Scan Mat display support
2. Calibration
3. Measurement
4. Scan history
5. Project memory
6. OpenSCAD starter generation
7. Maker Lab UI
8. Agentic engineering workflows

## Design Rule

Do not build advanced features on unstable vision outputs.

Raw, annotated, rectified, and calibrated outputs must come first.
