# Jarvis Phase 4 – Agentic Engineering Partner

## Goal
Phase 4 introduces controlled agentic workflows.

Jarvis should be able to plan, execute, verify, and summarize engineering tasks while keeping the human in control.

## Candidate Capabilities
- Multi-step task planning
- Local repo inspection
- Test execution
- Documentation updates
- Scan-to-model workflows
- Optional hardware automation
- Scheduled maintenance checks
- Local system monitoring

## Human Control Requirements
Jarvis must ask before:
- modifying source code
- deleting files
- pushing to GitHub
- running destructive commands
- controlling hardware motion
- sending external messages

## Success Criteria
Jarvis can perform useful engineering work safely, transparently, and locally.
