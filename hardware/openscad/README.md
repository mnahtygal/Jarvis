# Camera Mount Designs

## Logitech C920 Workbench Mount

`camera_mounts/logitech_c920/` contains the editable OpenSCAD source and STL
revision for the fixed overhead workbench camera:

```text
LogicamMountv1.scad
LogicamMountv1.stl
```

## Insta360 Link Mount History

`camera_mounts/inst360_link/` preserves the earlier Insta360 mount sources and
STL revisions:

```text
cameramount.scad
cameramountv2.scad
ShorterCameraMount.stl
ShorterCameraMountv2.stl
```

The Insta360 Link is now the `face` camera. These files remain useful design and
revision history but do not define the current Scan Mat camera.

Generated or revised mounts should retain editable OpenSCAD source, named
parameters, units, assumptions, and corresponding STL revision notes.
